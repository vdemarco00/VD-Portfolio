Archaeology Simulation Project

An educational digging/artifact inspecting simulation program for archaeology students.

This project was developed for the SNHU Online school, and is intended to be supplemental content for students studying archaeology.

This program realistically simulates digging for artifacts. The user skims their shovel along the dig site to slowly reveal what's buried underneath, and sifts through the soil that they dig up to find smaller artifacts.

The program runs in a web browser to make it accessible to most students. Most of my work on this project was fixing and adjusting unfinished or broken features, and optimizing the project to run on a web browser.
