PrimeFPS

An in-progress prototype FPS game inspired by the Metroid Prime series. Check it out here.

This goal of this project is to create an FPS game with solid, responsive keyboard & mouse controls with cool sci-fi weapons and abilities.

Current features of the project include a smooth character controller, a system for different weapon projectile types, picking up and throwing physics objects, and projectile collision exerting force on the objects hit.

Planned features: more projectile types that inflict varying amounts of damage and different types of damage, a modular system for creating interactions, maybe a grapple hook(?), and basic enemy AI.
